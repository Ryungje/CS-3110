1. OPAM housekeeping: We need to install ANISTerminal before running the game, so type in the following commands into terminal
  $ opam update
  $ opam upgrade
  $ opam upgrade
2. download blackjack.zip
3. change directory to the folder containing blackjack.zip
4. type the following command into terminal: 
  $ unzip blackjack.zip
  $ make build
  $ make play
5. Congratulations, you can now play the game from your terminal.
